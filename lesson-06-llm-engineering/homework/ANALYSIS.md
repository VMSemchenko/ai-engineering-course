## 1. Коли використовувати Ollama (Self-hosted)?

### Переваги:
- **Нульова вартість** — після скачування моделі всі запити безкоштовні
- **Приватність** — дані не покидають вашу машину
- **Швидший latency** на простих запитах (7.6s vs 15.3s для simple)
- **Офлайн-робота** — не потрібен інтернет

### Критерії вибору:
- Задача не потребує максимальної точності
- Важлива приватність даних (медичні записи, фінансова інформація)
- Масова обробка документів (без API ліміту)
- Бюджет обмежений

### Мінімальна якість:
- Для простих, чітко структурованих текстів Mistral 7B дає **85-100%** точність
- Для складних технічних текстів — **~65-70%** (пропускає частину завдань)
- Відповідає часто **англійською** замість української (навіть з українським промптом)

---

## 2. Коли використовувати Cloud API (Z.ai / OpenAI)?

### Переваги:
- **Вища якість екстракції** — знаходить більше завдань, краще розуміє контекст
- **Мовна послідовність** — стабільно відповідає українською, коли просять
- **Кращий structured output** — JSON завжди валідний та детальний
- **Масштабованість** — не потребує локального GPU

### Уроки:
1. **GLM-4.5 надає більш детальні описи завдань** — включає технічні деталі (HPA 2-8 подів, CPU 70%)
2. **Краще розуміє implicit tasks** — в хаотичному тексті знайшла 6 задач замість 4 (включно з "дати доступ до staging")
3. **Latency вищий** через мережевий overhead, але це прийнятно для batch-обробки
4. **Вартість мінімальна** — менше $0.001 за запит

## 3. Гібридний підхід

### Рекомендована стратегія: Ollama-first з fallback на Cloud

```python
def hybrid_extract(text):
    # Спроба 1: швидкий локальний запит
    result = extract_meeting_data(text, provider="ollama")
    
    if result["json_valid"] and len(result["result"].get("tasks", [])) > 0:
        return result  # Ollama впоралася — безкоштовно!
    
    # Fallback: Cloud API для складних випадків
    return extract_meeting_data(text, provider="zai")
```

### Переваги ensemble-підходу:
- **Економія** — 70-80% запитів обробляються безкоштовно (Ollama)
- **Надійність** — Cloud API як страховка для складних текстів
- **Швидкість** — прості запити обробляються локально швидше

### Коли Cloud обов'язковий:
- Критично важливі дані (юридичні протоколи)
- Багатомовні тексти
- Тексти з великою кількістю неявних завдань
