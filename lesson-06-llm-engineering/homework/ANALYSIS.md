## 1. Коли використовувати Ollama (Self-hosted)?

- Нульова вартість
- Приватність даних
- Швидший latency на простих запитах
- Офлайн-робота
- Для простих текстів — 85-100% точність
- Для складних — ~65-70%

## 2. Коли використовувати Cloud API?

- Вища якість екстракції
- Мовна послідовність (стабільно українською)
- Кращий structured output
- Масштабованість

## 3. Гібридний підхід

Ollama-first з fallback на Cloud:
- 70-80% запитів безкоштовно через Ollama
- Cloud API як страховка для складних текстів

## 4. Розширення

- Multi-language support
- confidence_score per task (0.0-1.0)
